# Typing Speed Test App : SpeedTypo

A full-stack typing test app built with:

- React + Vite
- Node.js + Express
- MongoDB
- Auth0 for authentication
- Chart.js for stats

## Features

- Google Login
- Timed typing test (30s)
- WPM, accuracy, and character count calculation
- Score tracking and graph display
- Responsive design

## Run Locally

### Frontend:

cd client
npm install
npm run dev

### Backend:

cd server
npm install
npm start
